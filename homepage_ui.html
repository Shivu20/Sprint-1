Homepage – UI

Description

As a Customer, I want to view the Home Page of the application with all required options relevant to the customer.

Acceptance Criteria

After logging in to the system using the customerId and password, the customer can view the home page with the following information on the screen:

  • At the right side, display: Welcome

Top Menu

  • Home
  • Booking Service
  • Tracking
  • Previous Booking
  • Contact Support
