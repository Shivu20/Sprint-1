Invoice Page

Description

As a customer/officer, I should be able to view the invoice once the payment is successful.

Acceptance Criteria

The invoice page needs to contain the below details in a read-only format with a print option:

  • Receiver Name
  • Receiver Address
  • Receiver Pin
  • Receiver Mobile
  • Parcel Weight (g)
  • Parcel Contents Description
  • Parcel Delivery Type
  • Parcel Packing Preference
  • Parcel Pickup Time
  • Parcel Dropoff Time
  • Parcel Service Cost
  • Parcel Payment Time
